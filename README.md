🚀 Features
IR Signal Reception  
Reads IR signals from the IR Remote Unit and decodes them.

Save to SD Card  
Stores captured IR codes as IR files on the SD card.

IR Signal Transmission  
Loads saved IR codes from the SD card and sends them through the IR transmitter.

On‑Device Menu Interface  
A simple Cardputer UI allows:

Recording IR codes

Listing saved codes

Sending selected codes

Pin Configuration

IR Receiver → G1

IR Transmitter → G2

🧰 Requirements
M5Stack Cardputer

M5Stack IR Remote Unit

MicroSD card
